Preguntas y Respuestas

1. ¿Qué es una red informática?

Una red informática es un conjunto de dispositivos electrónicos interconectados que comparten recursos y permiten la comunicación entre sí.

2. ¿Qué es una red LAN?

Una red LAN (Red de Área Local) es una red que se limita a un área geográfica pequeña, como una oficina, edificio o campus. Los dispositivos en una LAN pueden compartir recursos y comunicarse directamente entre sí.

3. ¿Qué es una red WAN?

Una red WAN (Red de Área Extensa) abarca una geografía más extensa, como ciudades, países o incluso continentes. Las WAN conectan redes LAN distantes y utilizan tecnologías como Internet y servicios de telecomunicaciones para facilitar la comunicación.

4. ¿Qué es Internet?

Internet es una red global de redes interconectadas que utilizan el protocolo TCP/IP para facilitar la comunicación y el intercambio de información entre dispositivos en todo el mundo.

5. ¿Qué es la Web?

La Web es un sistema de información global en Internet que permite acceder y compartir documentos multimedia mediante enlaces de hipertexto. La World Wide Web (WWW) es una parte esencial de Internet.

6. ¿Qué es WWW?

WWW significa World Wide Web y es un sistema de información en la Web que utiliza documentos en formato HTML interconectados mediante enlaces de hipertexto. Es un componente clave de Internet que permite la navegación y el acceso a recursos multimedia.

7. ¿Qué es el protocolo HTTP y HTTPS?

HTTP (Protocolo de Transferencia de Hipertexto) y HTTPS (HTTP Seguro) son protocolos de comunicación utilizados para la transferencia de datos en la Web. HTTPS utiliza una capa de seguridad adicional (SSL/TLS) para cifrar la información, proporcionando una conexión más segura.

8. Página web estática vs página web dinámica.

Una página web estática muestra contenido fijo y no cambia según la interacción del usuario. En cambio, una página web dinámica se genera en tiempo real y puede adaptarse según las acciones del usuario o datos provenientes de bases de datos.

9. CMS. Ejemplos. Ventajas e inconvenientes.

Un CMS (Sistema de Gestión de Contenidos) facilita la creación y gestión de contenido web. Ejemplos incluyen WordPress, Joomla y Drupal. Las ventajas incluyen facilidad de uso, mientras que los inconvenientes pueden incluir la necesidad de actualizaciones de seguridad regulares.

10. Proveedor de servicios de Internet. Ejemplos.

Proveedores de servicios de Internet (ISP) ofrecen acceso a Internet. Ejemplos incluyen Comcast, AT&T, y Verizon, que proporcionan servicios como banda ancha y conexión a la red.

11. Dirección IP. Diferencia entre una dirección IP privada y pública.

Una dirección IP identifica un dispositivo en una red. Las direcciones IP privadas son utilizadas en redes locales, mientras que las direcciones IP públicas son asignadas a dispositivos conectados directamente a Internet.

12. ¿Qué es el alojamiento web/hosting? Ejemplos.

El alojamiento web es el servicio que permite almacenar y acceder a los archivos de un sitio web. Ejemplos de proveedores de alojamiento web incluyen Bluehost, HostGator y SiteGround.
